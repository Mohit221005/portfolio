# Mohit Kumar Portfolio

A modern, animated, and responsive portfolio website built with React, showcasing my skills, projects, and contact information.

## 🚀 Features
- Visually appealing header with custom background and profile image
- Animated transitions using Framer Motion
- Responsive design for all devices
- Skills section with progress bars
- Projects section with images, tech stack, and links
- Contact form and contact information

## 🛠️ Getting Started

### Prerequisites
- Node.js (v14 or above)
- npm or yarn

### Installation
```bash
git clone https://github.com/Mohit221005/portfolio.git
cd portfolio
npm install
```

### Running Locally
```bash
npm start
```
Visit [http://localhost:3000](http://localhost:3000) to view the site.

## ✨ Usage
- Update your profile image at `public/images/profile.jpg`.
- Update the header background at `public/images/headerbg.jpg`.
- Edit your skills, projects, and contact info in the respective component files in `src/components/`.
